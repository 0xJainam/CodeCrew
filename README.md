# CodeCrew

CodeCrew is a collaborative code editor inspired by modern IDE workflows, built with Next.js, React, TypeScript, Tailwind CSS, and Monaco Editor.

## Features

- Monaco-powered code editor
- Multi-file explorer
- Create, rename, and delete files
- Tab-based file navigation
- Language switching support
- IDE-style workspace layout
- Responsive landing page
- Bottom output/terminal panel UI
- Reusable component-based architecture

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Monaco Editor
- Lucide React

## Upcoming Features

- Real-time collaboration with Socket.IO
- Room creation and room joining
- Live presence indicators
- Shared file synchronization
- Collaborative cursor tracking
- Code execution backend

## Run Locally

Clone the repository:

```bash
git clone https://github.com/yourusername/codecrew.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
